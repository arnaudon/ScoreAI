## ScoreAI


Python app for playing music from pdfs with the help of AI.


# main features to implement
- manually add pdfs to a database with entries like title, composer, period, edition, etc...
- show a list of available pdfs, sort/classify by composer, period, etc...
- display a pdf in a pdf viewer, easy to navigate left/right (https://github.com/lfoppiano/streamlit-pdf-viewer)
- add an AI assistant to (with pydantic AI):
  - choose a piece to play
  - sugest other similar pieces, possibly add from imslp to the db directly
  - to provide context on the piece (history, how to play, etc...)
  - show youtube video of performance (different types, educational, best interpreters, etc...)
